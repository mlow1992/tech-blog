# Tech-Blog

## Purpose
This app is a CMS-style blog site where developers can publish their posts and comment on other developers' posts. Additionally, users can upvote 

## Built With

- JavaScript
- bcrypt
- Sequelize
- Express.js
- SQL
- CSS
- Heroku
